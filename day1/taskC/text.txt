C. Йода.Начало
Time limit	1 секунда
Memory limit	64Mb
Input	stdin or input.txt
Output	stdout or output.txt

Legend
Йода с детства начинал переставлять слова в предложении в причудливом порядке. Напишите программу, которая будет имитировать его деятельность.
Вам даны следующие переменные (не забудьте включить их в свою программу):
word1 = "имя"
word2 = "твое"
word3 = "мне"
word4 = "знакомо"
Ваша программа должна выводить 3 строки, используя исключительно переменные, приведенные выше. Должны получиться строки : "знакомо мне твое имя"
"мне имя знакомо твое"
"твое знакомо имя мне".

Input format
На вход программе ничего не подается.

Output format
знакомо мне твое имя
мне имя знакомо твое
твое знакомо имя мне
